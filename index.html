Diseño de página web de presentación personal (github)

En mi primer proyecto, desarrollé una página web de presentación personal, en la cual me introduzco y detallo mis habilidades y tecnologías que manejaba en ese momento. Utilicé HTML, CSS y JavaScript para construir la página, lo que me permitió aprender y aplicar conceptos fundamentales de ECMAScript. Este proyecto fue una excelente oportunidad para fortalecer mis habilidades técnicas y establecer una base sólida en el desarrollo web

2024-10

Turnos bancarios (github)

Desarrollé una aplicación web Full Stack utilizando TypeScript, Express, SQL y React. El backend se construyó con TypeScript, Express, SQL y TypeORM. El frontend se desarrolló con React y TypeScript. La aplicación sigue una arquitectura Full Stack, con el backend manejando la lógica de negocio y el acceso a datos, y el frontend proporcionando la interfaz de usuario. Desarrollo de Backend para E-commerce (github)

Desarrollé el backend de una aplicación de E-commerce utilizando NestJS, TypeORM y Docker. El backend fue desarrollado con NestJS y TypeScript, lo que permitió una arquitectura modular y organizada.
